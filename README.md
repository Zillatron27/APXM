# APXM — Mobile PrUn Interface

Mobile-optimized interface for [Prosperous Universe](https://prosperousuniverse.com) (apex.prosperousuniverse.com).

## Development

```bash
# Chrome
pnpm dev

# Firefox
pnpm dev:firefox
```

## Build

```bash
# Chrome
pnpm build

# Firefox
pnpm build:firefox
```

## Package

```bash
pnpm zip
pnpm zip:firefox
```
